Add contributions to the Jacobian from derivates calculated using the output of this sub model. 

Derivatives are assumed to be of the form $ frac{dy}{dt} = A*S $, where $A$ is the value passed to this function as {\bfseries base\+\_\+val} and $S$ is the sub-\/model parameter used in the calculation. The row of the Jacobian should correspond to $frac{dy'}{dx}$, where for each element $x$, on the row, this function will add $A*frac{dS}{dx}$.
